참고:

- PR 문서 양식은 `.github/PULL_REQUEST_TEMPLATE.md`에 있다.

- Git 기준: 대상 브랜치(보통 `main`)와의 차이(`git log main..HEAD`, `git diff main...HEAD`)로 실제 변경·커밋을 파악한다.

{{input}}

요구사항:

- 템플릿의 **섹션 구조·체크리스트**를 그대로 유지한 채 본문을 채운다.
- **관련 이슈**는 `close #번호`는 실제 번호가 있을 때만; 없으면 “없음” 또는 생략 규칙을 안내한다.
